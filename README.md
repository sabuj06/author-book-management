# Book & Author Management System (Laravel)

This is a simple Laravel backend application built to manage Authors and their Books.
The project demonstrates basic CRUD operations, proper model relationships, and request validation.

## Features
- Author management (Create, Read, Update, Delete)
- Book management (Create, Update, Delete)
- One-to-Many relationship between Authors and Books
- Form Request Validation
- RESTful API structure

## Tech Stack
- Laravel
- PHP 8.1+
- MySQL
- Composer

## Requirements
- PHP >= 8.1
- Composer
- MySQL
